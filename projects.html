Plant Disease Detection Organic Fertilizer Recomendation System

Duration: May 2024 – October 2024

My Role:worked in the front end part and Documentation

Technologies used: Python, Machine Learning, CNN, OpenCV, TensorFlow, Flask, HTML, CSS.

This project focuses on identifying plant diseases at an early stage using image processing and machine learning techniques. The system analyzes images of plant leaves to detect visible disease symptoms accurately and classifies the type of disease affecting the plant. Early detection helps farmers reduce crop loss and improve agricultural productivity.Based on the detected disease and crop type, the system recommends suitable organic fertilizers and eco-friendly treatment methods. These recommendations aim to improve soil health, reduce chemical dependency, and promote sustainable farming practices. The project integrates technology with agriculture to support smart decision-making, enhance crop yield, and encourage environmentally responsible farming.

Fake Currency Detection

Duration: November 2024 – April 2025

My Role: worked in the front end part and Documentation

Technologies used: Python, Machine Learning, CNN, OpenCV, TensorFlow, Flask, HTML, CSS.

This project aims to detect counterfeit currency notes using machine learning and image processing techniques. The system analyzes key visual features of currency notes such as texture, color patterns, security marks, and watermark regions to distinguish between genuine and fake notes with high accuracy. Images of currency notes are preprocessed to enhance quality and extract important features. A machine learning model is trained on these features to classify the currency as real or counterfeit. The system helps in preventing financial fraud, reduces manual verification effort, and provides a fast, reliable, and automated solution for currency authentication.

Stachi Art Web Page

Duration: practice project

My Role: Own project

Technologies used: Html, Css, Js, Bootstarp
